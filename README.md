# CodeAlpha Language Translation Tool

This project is developed as part of the CodeAlpha Artificial Intelligence Internship.

## Task Description
A language translation tool that allows users to:
- Enter text
- Select source and target languages
- Translate text using Google Translate API
- View translated output clearly
- Use copy-friendly output and text-to-speech for better usability

## Technologies Used
- Python
- Googletrans (Google Translate API)
- gTTS (Text-to-Speech)
- ipywidgets
- Google Colab

## How to Run
1. Open the notebook in Google Colab
2. Run cells sequentially
3. Enter text and select languages
4. Click Translate
